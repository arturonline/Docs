# RustUP!

`rustup` is a command line tool for managing Rust versions and associated tools.

## Updating and Uninstalling

After youâ€™ve installed Rust via `rustup`, updating to the latest version is easy. From your shell, run the following update script:

```bash
rustup update
```

To uninstall Rust and `rustup`, run the following uninstall script from your shell:

```bash
rustup self uninstall
```

## Local Documentation

The installer also includes a copy of the book and the documentation locally, so you can read it offline:

```bash
# book:
rustup docs --book

# API:
Run rustup doc
```
