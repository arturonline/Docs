# Comments

## Help comments

Syntax for Comment-Based Help in Functions:

```powershell
<#
.SYNOPSIS
.DESCRIPTION
.PARAMETER x
.PARAMETER y
.INPUTS
.OUTPUTS
.EXAMPLE
.EXAMPLE
.EXAMPLE
.EXAMPLE
.LINK
.LINK
.NOTES
#>
```

For example, the Description keyword precedes a description of a function or script:

```powershell
<#
.Description
Get-Function displays the name and syntax of all functions in the session.
#>
```
