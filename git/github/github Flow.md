# Github Workflow

1. Download a local copy of the repository:

    ```bash
    git clone https://github.com/arturonline/exam-70483.git
    ```

2. After finishing job, upload changes:

    ```bash
    git push origin master

    # origin = git remote -v
    # master = git branch

    ```

3. To get latest changes

    ```bash
    git pull origin master
    ```
